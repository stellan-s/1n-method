  - Includes key terms section with definitions

### Changed

## [1.0.0] - 2026-01-25

### Added

- Initial release of the 1..n Method specification
- Core documentation structure with 8 foundational documents:
  - Summary: One-page overview of the method
  - Foundations: Seven observable constraints that shape the method
  - Method Principles: Seven core operating principles
  - Decision Flow: Mermaid diagram and explanation of how work moves through the system
  - What It Does Not Solve: Explicit list of 10 non-goals
  - Leadership and "When?": Guidance on answering timeline questions without false precision
  - Metrics and Progress: Framework for measuring reality over activity
  - Glossary: Definitions of key terms and concepts
- MIT License
- README with quick start guide and versioning approach
- Decision flow visualization using Mermaid flowchart

- **Sequential Priority**: Roadmap language using "now, next, later" instead of dates
- **Conditional Commitment**: Honest framing of future work with stated assumptions
